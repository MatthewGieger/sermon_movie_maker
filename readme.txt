SERMON MOVIE MAKER
------------------
Creator: Matthew Gieger
Date Started: 11/13/2016


** What is it? **
A python script that converts an mp3 file into a mp4 video file with a sermon's audio and a static image along with sermon details as an overlay to the image. These videos can then be uploaded to the church website and/or the church facebook page.


** How to Use it **
When prompted, enter the Sermon Title, the Speaker, and the Date. The program will then get to work on exporting the mp4 file in the location set in the settings file.


** Requirements **

ffmpeg
Python 2.7
	mutagen
